# Telegram Channel Copier (Posts + Comments)

Скрипт копирует новые посты из публичного Telegram-канала в ваш тестовый канал:
- текст + форматирование
- вложения (фото/видео/документы), включая альбомы
- комментарии к постам (текст/стикеры/медиа) → публикуются как комментарии под соответствующим постом в вашем канале

Синхронизация инкрементальная: состояние хранится в `.env` через `TG_LAST_SEEN_ID`.

---

## Требования
- Python 3.10+
- Telegram API credentials
- StringSession вашего аккаунта
- В **целевом канале** должны быть включены комментарии (привязан discussion-чaт), если хотите копировать комментарии.
